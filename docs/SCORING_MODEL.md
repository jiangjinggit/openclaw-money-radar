# 项目评分模型

## 当前 5 个核心评分维度
### 1. moneyProbability
赚钱概率：是否容易在 30 天内拿到第一笔钱。

### 2. launchSpeed
启动速度：是否能在 7-14 天内做出 MVP。

### 3. timeFriendliness
时间友好度：是否适合上班之外推进，是否低同步、低服务消耗。

### 4. contentPotential
内容传播性：是否适合写教程、案例、复盘、榜单，带来高意向流量。

### 5. expansionPotential
可复制扩张性：是否能从单点产品扩成订阅、团队版或更长期产品。

---

## 当前使用原则
- `T1` 不代表所有维度都最高，而是综合最适合当前阶段优先做。
- 短期现金流项目更看重 `moneyProbability`、`launchSpeed`、`timeFriendliness`。
- 长期主壳项目更看重 `contentPotential`、`expansionPotential`、留存逻辑。
