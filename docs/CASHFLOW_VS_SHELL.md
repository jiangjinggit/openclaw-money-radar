# 短期现金流 vs 长期主壳

## 短期现金流优先
### 安全审计 / 安全体检 / 成本治理报告器
更适合：
- 先拿第一笔钱
- 先做低同步交付
- 先用标准报告和修复建议证明价值

为什么更强：
- 付费意愿更直接
- 更接近损失预防
- 7-14 天内可做 MVP

---

## 长期主壳优先
### Monitor Lite / Agent Ops Lite
更适合：
- 承接持续运行层
- 做长期留存和订阅
- 逐步升级成多节点、团队版、主壳产品

为什么更强：
- 成本、告警、节点健康、错误日志是长期需求
- 留存逻辑更自然
- 更适合长期层，而不是短期首单层

---

## 当前总判断
- **先收钱**：安全体检
- **长期承接**：monitor shell
