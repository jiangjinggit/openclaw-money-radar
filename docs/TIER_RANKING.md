# OpenClaw 项目机会梯队（T1 / T2 / T3）

## T1：当前最值得优先做
### 1. OpenClaw 安全审计 / 安全体检 / 成本治理报告器
- 最接近短期 first money
- 真实痛点强
- 适合异步交付
- 可产品化、可模板化、可内容化

### 2. OpenClaw Monitor Lite / Agent Ops Lite
- 更适合长期 retention shell
- 长期价值强
- 适合承接持续运行层
- 但首单变现速度略慢于安全体检

---

## T2：值得做，但优先级略低
### 1. OpenClaw 行业模板包 / 工作流商店
- 上线快
- 适合数字产品化
- 但痛感和付费紧迫性弱于安全体检

### 2. OpenClaw 知识 / 培训 / 方法论产品
- 适合内容沉淀和长期品牌资产
- 但不是最直接的 first money 切口

---

## T3：当前不优先
### 1. 重咨询型企业方案
- 同步成本高
- 交付重
- 不适合当前阶段优先做

### 2. 一上来做大而全企业平台
- 开发周期长
- 解释成本高
- 不适合独立开发者先跑通首单
